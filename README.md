# Social Media Automator

Complete automation + testing hub for social media across all platforms (Facebook, Instagram, LinkedIn, Twitter, TikTok).

**Features:**
- ✅ Automated testing of all social platforms
- ✅ GHL conversation monitoring and ingestion
- ✅ Claude-powered auto-responses to comments, DMs, messages
- ✅ Keyword-triggered smart responses
- ✅ 5-minute auto-response checks
- ✅ Hourly platform health tests
- ✅ Integration with Library Control for test tracking
- ✅ Webhook support for real-time inbound messages

## API Endpoints

```
POST /api/test-social           → Run social platform tests
POST /api/auto-respond          → Process and auto-respond to conversations
GET  /api/test-results          → View test results history
GET  /api/response-log          → View auto-response activity
GET  /api/status                → Service status
POST /api/webhook/ghl           → GHL webhook receiver (comments/DMs)
```

## Automatic Schedules

- **Auto-response checks**: Every 5 minutes (fetch GHL conversations, respond if needed)
- **Platform tests**: Every hour (test all 5 social platforms)

## Environment Variables

Required:
- `ANTHROPIC_API_KEY` — Claude API key (for intelligent responses)
- `GHL_API_KEY` — GoHighLevel API token
- `GHL_LOCATION_ID` — GHL location ID

Optional:
- `SOCIAL_CAMPAIGN_URL` — rei-social-campaign service URL
- `LIBRARY_URL` — Library Control URL
- `LIBRARY_PASSWORD` — Library Control password
- `PORT` — Server port (default: 3000)

## Auto-Response Logic

Responds to conversations that:
1. Contain keywords: buy, sell, deal, property, real estate, investment, partner, wholesale
2. Haven't been responded to yet
3. Are not own messages

Each response is:
- Generated by Claude
- Personalized with contact name
- Includes call-to-action
- Signed with agent name
- Logged for audit trail
